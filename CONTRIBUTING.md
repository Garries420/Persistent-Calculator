# Contributing

Feedback, bug reports, and focused feature suggestions are welcome.

Before opening an issue:

1. Confirm the problem still occurs in the latest public release.
2. Check existing issues for the same report.
3. Do not attach calculation-history files or screenshots containing personal information unless they are carefully sanitized.

Persistent Calculator intentionally supports Standard mode only today. Requests for Scientific, Graphing, Programmer, Date calculation, or individual converter modes help determine which additions may be worth building later.

Security vulnerabilities should be submitted through GitHub's private security-advisory feature as described in [SECURITY.md](SECURITY.md), not through a public issue.
